## 🚀 Últimas Actualizaciones / Latest Updates

### ✅ **Header Completado / Header Completed**
Se ha finalizado la implementación y rediseño de la barra de navegación (**Header**).
* **Diseño:** Estilo "Angelical" con efecto Glassmorphism (Cristal esmerilado).
* **Responsive:** Adaptación completa para dispositivos móviles con menú lateral animado y lógica de scroll.
* **Tecnología:** Migración de estilos CSS inline a clases de **TailwindCSS** para mejor rendimiento y mantenimiento.
* **Funcionalidad:** Dropdowns animados con `framer-motion` y navegación fluida.


### 🎠 **Carrusel Completado / Carrousel Completed**
Se ha finalizado la implementacion y diseño del carrousel de la pagina CAFSA (**Carrousel**).
* **🎨 Diseño UI/UX Refinado:** Estética minimalista que prioriza el contenido. Colores y contrastes ajustados para eliminar el "ruido visual" y mantener la coherencia con la identidad de marca.
* **📱 100% Responsive:** Adaptabilidad fluida desde dispositivos móviles hasta pantallas de escritorio (Full-width & Full-height).
* **✨ Animaciones Dinámicas:** Transiciones suaves de imágenes y textos orquestadas con **Framer Motion**.
* **⚡ Funcionalidad:** Reproducción automática (Autoplay), navegación manual intuitiva e indicadores de estado.

### ✅ **Sobre Nosotros completado / AboutUs Completed**
Se ha finalizado el modulo "sobre nosotros" y la implementacion de un modal que te dice los horarios disponibles de los servicios
* **📱 100% Responsive:** Adaptabilidad fluida y para nada brusca para dispositivos moviles. Layout adaptativo que utiliza Grid CSS (2 columnas en escritorio, 1 en móvil).
* **Animaciones:** Entrada suave de elementos utilizando framer-motion.
* **Estética Visual:** Uso de tarjetas decorativas, gradientes y la paleta de colores corporativa.
* **Interacción:** Botón "Ver horarios" que activa el estado del modal.

#### Modal de Horarios 

* **Overlay Inmersivo:** Fondo oscuro con efecto backdrop-blur (desenfoque) para centrar la atención.
* **Posicionamiento:** Centrado absoluto (fixed) garantizado mediante CSS, independiente del scroll de la página.
* **Animaciones de Entrada/Salida:** Efecto spring (rebote suave) al abrirse y cerrarse.
* **Cierre Intuitivo:** Se puede cerrar mediante el botón dedicado o haciendo clic fuera del contenido (en el overlay).

## 🇪🇸 Sobre el Proyecto

### **Comunidad de Adoración Familia Saludable (CAFSA)**

Este es el sitio web oficial de la iglesia **CAFSA**. El objetivo principal de este portal es brindar información clara y armoniosa sobre:

* 🏛️ La Iglesia y su visión.
* 📅 Actividades y eventos.
* ✨ Ministerios y servicios.
* 👥 El equipo y las personas que conforman la comunidad.

Aunque el sitio web es **meramente informativo**, cuenta con **funcionalidades dinámicas** modernas para ofrecer una experiencia de usuario fluida y agradable.

## 🇺🇸 About the Project

### **Healthy Family Worship Community (CAFSA)**

This is the official website of the **CAFSA** church. The main goal of this portal is to provide clear and harmonious information about:

## 🛠️ Stack Tecnológico / Tech Stack

Este proyecto ha sido construido utilizando tecnologías modernas de alto rendimiento:
